# Cursor IDE Setup Assignment

## Tools Installed
- Cursor IDE
- Claude Code for VS Code extension
- Codex – OpenAI's coding agent extension
- Git
- GitHub

## Steps Completed
1. Installed Cursor IDE.
2. Installed the Claude Code for VS Code extension and configured it.
3. Installed the Codex extension and configured it.
4. Created a public GitHub repository.
5. Opened the repository in Cursor.
6. Created and updated the `README.md` file.
7. Installed Git for Windows to enable source control.

## Issues Encountered
- I initially had trouble locating the Extensions marketplace in Cursor.
- Git was not installed on my system, so Cursor could not use source control features. I installed Git and restarted Cursor to resolve the issue.

## Outcome
The required tools were installed successfully, the repository was created, and the setup process was documented in this README.
